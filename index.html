What is Plumber?

Plumber automates the deployment process of applications or service from the configuration, tests to the deployment.

How to build

Requirements

  • Go 1.3 or greater

How to Build

You can build plumber with the following commands.

$git clone https://github.com/recruit-tech/plumber.git
$cd plumber
$./build

How to contribute

We welcome any contributions through Github pull requests. When you make changes such as new features and the add the tests, please run test before throw the pull reqest. You can run test with the test.sh script.

$sh test.sh
